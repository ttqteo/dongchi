# Đồng Chi

Chia tiền nhóm và quay số chọn người, ngay trên trình duyệt. Không cần đăng ký, không lưu dữ liệu cá nhân.

## Tính năng

- **Chia tiền**: chia đều (tổng ÷ số người, làm tròn 1.000đ) hoặc theo món (từng người + phí + khuyến mãi). Xuất kết quả dạng text hoặc CSV.
- **Quay số**: vòng quay chọn ngẫu nhiên người thực hiện nhiệm vụ.
- Dark mode, lưu `localStorage`, giao diện mobile-first.

## Phát triển

```bash
npm install
npm run dev      # http://localhost:3000
npm run build
```
